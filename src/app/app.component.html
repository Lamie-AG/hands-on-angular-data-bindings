Angular Data Binding Example

In this example we will look at data bindings which allow us to reference data from our .ts files in our html code. Binding data is really important since it allows for dynamic content. We can even bind data from one component to another.

We can either use a one-way binding by using [property]="someVariable" or [(ngModel)]="someVariable" for a two-way binding.
For binding properties we use [...] but whenever we want to bind to an event we need (...) and that's why we have to use both [(...)] for two-way bindings.

Type Syntax
String Interpolation {{"{{expression}}"}}
One-way [target]="expression"
Event (target)="expression"
Two-way [(target)]="expression"
